# Tilia
Developed using React, Golang, and GraphQL. 


### Stack

```
frontend:
React

mobile:
firebase
react-native

Backend:
Golang
Graphql
Mongodb

Design:
Figma
```

### How to Run

Run the server

```
cd into app/server/src/app

export GOPATH=[insert path to your project]/app/server

dep init

dep ensure

go run main.go
```

Run the application

```
cd app/client

npm run dev
```
